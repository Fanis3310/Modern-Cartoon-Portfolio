🚀 Fanis Moutso - Personal Portfolio
A modern, interactive, and fully responsive personal portfolio website showcasing my journey as a Junior Web Developer. This single-page application (SPA) is built with vanilla web technologies and features unique interactive elements, including a fully functional web-based terminal and an interactive journey map.

✨ Features
Terminal Interface (

/terminal.html
): A playable, interactive command-line interface that allows users to type commands (like help, about, skills, projects) to navigate the portfolio in a unique way. Features ASCII art, a matrix digital rain effect, and Easter eggs!
Interactive Journey Map: A visual mapped timeline of my professional experience across Greece, tracking my career from Technical Support to Web Development.
Modern UI/UX: Features glassmorphism, smooth scrolling, CSS grid/flexbox layouts, and dynamic theme switching.
Neo-Brutalism Support: Includes an experimental neo-brutalism CSS styles variant (neo-styles.css).
Fully Responsive: Optimized for all devices, from large desktop monitors to mobile phones.
SEO Optimized: Includes meta tags, semantic HTML5, and a sitemap.xml.
🛠️ Built With
HTML5: Semantic and accessible structure.
CSS3: Custom vanilla CSS with CSS variables for theming, animations, and responsive design.
JavaScript (ES6+): Vanilla JS handling the terminal logic, dynamic data rendering, modal toggles, and map interactivity.
Leaflet.js: Used for the interactive geographic journey map.
