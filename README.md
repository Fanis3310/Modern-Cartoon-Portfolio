## ✨ Features

- **💻 Terminal Interface (`/terminal.html`)**  
  A playable, interactive command-line interface that allows users to type commands (like `help`, `about`, `skills`, `projects`) to navigate the portfolio in a unique way. Features ASCII art, a matrix digital rain effect, and Easter eggs!
  
- **🗺️ Interactive Journey Map**  
  A visual mapped timeline of my professional experience across Greece, tracking my career from Technical Support to Web Development.

- **🎨 Modern UI/UX**  
  Features glassmorphism, smooth scrolling, CSS grid/flexbox layouts, and dynamic theme switching. Includes an experimental neo-brutalism CSS styles variant (`neo-styles.css`) and a Pirate Theme.

- **📱 Fully Responsive**  
  Optimized for all devices, from large desktop monitors to mobile phones.

- **🚀 SEO Optimized**  
  Includes meta tags, semantic HTML5, and a `sitemap.xml`.

## 🛠️ Tech Stack

### Core Technologies
- **HTML5**: Semantic and accessible structure.
- **CSS3**: Custom vanilla CSS with CSS variables for theming, animations, and responsive design. 
- **JavaScript (ES6+)**: Vanilla JS handling the terminal logic, dynamic data rendering, modal toggles, and map interactivity.

### Libraries & Tools
- **Leaflet.js**: Used for the interactive geographic journey map.

## 🚀 Getting Started

To run this project locally, simply clone the repository and open `index.html` in your browser. No build steps or package managers are required!

```bash
# Clone the repository
git clone https://github.com/Fanis3310/Modern-Cartoon-Portfolio.git

# Navigate to the project directory
cd Modern-Cartoon-Portfolio/fanismoutso.me

# Open index.html in your default browser
open index.html
```
